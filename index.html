Set up Python on your computer

1. Set up Python

We're going to use Python 3.3 for this workshop.

Python is available for:

  • Windows bit.ly/1mS4rtB
  • Mac OS X bit.ly/1lHStCH
  • Linux bit.ly/1dZrSie

Get from python.org.

PRO TIP: Python 2 and Python 3 are not the same - pick one and stick to it!

2. Prepare a Text Editor

Install a good text editor for writing and saving Python code during the workshop.

Recommendations:

  • Windows: Notepad++ bit.ly/1oCavbz
  • Mac OS X: Smultron bit.ly/1lHSy9w
  • Linux: Gedit bit.ly/1gy5Hde
PRO TIP: You need to set your editor to use spaces to indent.

3. Starting and exiting Python

Open a command prompt:

  • Windows: Start... Search... cmd
  • OS X/Linux: Start the 'Terminal' application

To start Python, type python3. You should see something a bit like this: 

Python 3.3.4 (default, Apr  6 2014, 10:27:50)
[GCC 4.2.1 Compatible Apple LLVM 5.1 (clang-503.0.38)] on darwin
Type "help", "copyright", "credits" or "license" for more information.
>>>

Exit the Python prompt by typing exit(). Now you're back at the command prompt.

4. Running Python code from a file

Start your text editor.

Add the following to a new, blank text file:

print("Hello World!")

Save the file as hello.py in your Desktop directory. The .py extension indicates the file contains Python code.

Start a command prompt and cd Desktop to move to the directory where we saved your file.

python3 hello.py should show:

Hello World!

Making choices

Use if statements to make decisions and conditionally execute code.

    print("Christmas colour!")
elif colour == "black" or colour == "orange":
    print("Halloween colour!")
elif colour == "pink":
    print("Valentine's Day colour!")
else:
    print("Some other colour!")

New Stuff

Lists []

Lists

Store data where order matters.

>>> my_list = []
>>> my_list
[]
>>> your_list = ["a", "b", "c", 1, 2, 3]
>>> your_list
['a', 'b', 'c', 1, 2, 3]

>>> type(my_list)
<class 'list'>

Access and adding elements

Indexed starting with 0.

my_list.append('new') to add to the end.

my_list.insert(0, 'new') to insert at the start.

my_list[0] = 'new' to change the first element.

len(my_list) to get the length.

'thing' in my_list to check containment.

Access and Slicing

my_list[0] to get the first element.

my_list[-1] to get the last element.

my_list[:3] to get the first 3 elements.

my_list[3:] to get everything after the first 3 elements.

Strings are a lot like Lists

>>> my_string = "Hello World"
>>> my_string[0]
'H'
>>> my_string[:5]
'Hello'
>>> my_string[6:]
'World'
>>> my_string = my_string[:6] + "Jessica"
>>> my_string
'Hello Jessica'

Big difference is you can't change strings.

Dictionaries {}

Dictionaries

Use dictionaries to store key/value pairs.

Dictionaries do not guarantee ordering.

A given key can only have one value, but different keys can have the same value.

>>> my_dict = {}
>>> my_dict
{}
>>> your_dict = {"Alice" : "chocolate", "Bob" : "strawberry", "Cara" : "mint chip"}
>>> your_dict
{'Bob': 'strawberry', 'Cara': 'mint chip', 'Alice': 'chocolate'}

>>> type(my_dict)
<class 'dict'>

Access and adding elements

your_dict["Alice"] to get the value for a key.

your_dict["Dora"] = "vanilla" to set or replace a key & value.

'Alice' in your_dict to check containment (only keys!).

len(your_dict) to get the number of items.

your_dict.get("Eve") to get the value for a key, without throwing an error if it doesn't exist.

Access and adding elements

your_dict.keys() returns all the keys.

your_dict.values() returns all the values.

They don't return lists, but you can convert it to a list using list():

>>> your_dict = {"Alice" : "chocolate", "Bob" : "strawberry", "Cara" : "mint chip"}
>>> your_dict.keys()
dict_keys('Bob', 'Cara', 'Alice')
>>> your_dict.values()
dict_keys('chocolate', 'strawberry', 'mint chip')
>>> list(your_dict.keys())
['Bob', 'Cara', 'Alice']
>>> list(your_dict.values())
['chocolate', 'strawberry', 'mint chip']

Loops

For loops

Use a for loop to do something to every element in a list.

>>> names = ["Jessica", "Adam", "Liz"]
>>> for name in names:
...     print(name)
...
Jessica
Adam
Liz

if statements inside for loops

Use to build up a list.

>>> vowel_names = []
>>> for name in ["Alice", "Bob", "Cassie", "Deb", "Ellen"]:
...     if name[0] in "AEIOU":
...         vowel_names.append(name)
...
>>> print(vowel_names)
['Alice', 'Ellen']

Using a counter

Create a variable and update it during a loop.

>>> prices = [1.5, 2.35, 5.99, 16.49]
>>> total = 0
>>> for price in prices:
...     total = total + price
...
>>> total
26.33

for loops inside for loops

>>> letters = ["a", "b", "c"]
>>> numbers = [1, 2, 3]
>>> for letter in letters:
...     for number in numbers:
...         print(letter * number)
...
a
aa
aaa
b
bb
bbb
c
cc
ccc

Order of your loops matters!

Useful list and loop functions

my_list.sort() to sort a list.

min(my_list) and max(my_list) to get extremes.

range() to generate a range of numbers: 

>>> range(5)
[0, 1, 2, 3, 4]

>>> what = input("Please type something? ")

Please type something? Hello

Modules

Modules

Don't write it twice! Modules allow us to organise and re-use code.

Don't write it at all! Python has 'batteries included', and there are thousands of modules for every conceivable purpose.

Each .py file is a module.

Imports

Use import to access a module.

>>> import random
>>> random.randint(1, 10)
5
>>> random.choice(["red", "green", "blue", "yellow"])
'green'

len(), type(), min(), max(), range(), str() are all "builtins" and don't need importing.
